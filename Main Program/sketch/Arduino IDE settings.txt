UNDER BOARDS MANAGER:
	INSTALL esp32 by Espressif Systems 2.0.xx ONLY e.g 2.0.17 (Not 3.x) (Else you'll get the timerAlarm error)
	https://github.com/espressif/arduino-esp32

Board: "DOIT ESP32 DEVKIT V1"

Install Required Libraries:

	1.	Adafruit SSD1306 2.5.11 by Adafruit 
		https://github.com/adafruit/Adafruit_SSD1306

	2.	Adafruit HTU21DF Library 1.1.2 by Adafruit 
		https://github.com/adafruit/Adafruit_HTU21DF_Library

	3.	IRremoteESP8266 2.8.6 by crankyoldgit 
		https://github.com/crankyoldgit/IRremoteESP8266

Serial Monitor: 57600 baud
